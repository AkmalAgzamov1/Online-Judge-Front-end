# Online Judge Frontend

This is the frontend part of the Online Judge project.  
It is built with React + Vite.

## Features

- User-friendly interface
- Problem list page
- Problem details page
- Code submission page
- Connection with backend API
- Fast development with Vite

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Axios / Fetch API

## Installation
```bash
1. Clone the repository:
https://github.com/AkmalAgzamov1/Online-Judge-Front-end

2. Go to project folder
cd Online-Judge-Front-end

3. install dependencies
npm install

4. Start the development server
npm rund dev

The project will usually run on
http://localhost:5173
